# File Handling Assignment

This project demonstrates how to read, write, and handle file errors in Python.

## Files

### 1. task1.py
- **Goal:** Open `sample.txt` and print its content.
- **Feature:** If the file is missing, it catches the error and prints a friendly message instead of crashing.

### 2. task2.py
- **Goal:** Create and modify `output.txt`.
- **Steps:**
  1. Asks user for input -> Writes to file.
  2. Asks for more input -> Appends to file.
  3. Reads the file back to verify data.
